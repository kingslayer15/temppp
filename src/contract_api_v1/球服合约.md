## 交易接口

#### 1.开启第二批销售
```JavaScript
    function newRound() public onlyRole(DEFAULT_ADMIN_ROLE);
    说明: 需要管理员DEFAULT_ADMIN_ROLE权限

```

#### 2.铸造
```javascript
    function mint(address to) external onlyRole(MINTER_ROLE);
    说明: 需要铸造管理员MINTER_ROLE权限

    入参：
        to： token接收地址
```

#### 3.批量铸造
```javascript
    function batchMint(address[] memory tos) external onlyRole(MINTER_ROLE);
    说明: 需要铸造管理员MINTER_ROLE权限

    入参：
        tos： token接收地址, 数组
```

#### 4.空投(官网铸造)
```javascript
    function airdrop(address to,uint256 salt, uint256 deadTime, bytes32 _hash,uint8 v,bytes32 r,bytes32 s) external;
    说明: 需要后台签名

    入参:
        to: 接收空投的地址
        salt: 后台返回的盐值
        deadTime：过期时间
        _hash: to + salt + deadTime 的hash串
        v,r,s: 签名
```


## 查询接口

#### 1.查询地址所有球服nft
```JavaScript
    function tokensOfOwner(address _owner) external view returns (uint256[] memory);
    
    入参:
        _owner: 查询地址
    返回:
        1. uint256[]: 拥有的tokenId, 数组
```

#### 2.查询两个批次的铸造数量
```javascript
    function numberOfTwoRounds() public view returns (uint256 round1Amount, uint256 round2Amount);

    返回：
        1. uint256 round1Amount: 批次1的铸造数量
        2. uint256 round2Amount: 批次2的铸造数量
```
